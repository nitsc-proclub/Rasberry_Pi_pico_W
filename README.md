# Rasberry Pi pico W

GitHub Pages:
https://nitsc-proclub.github.io/Rasberry_Pi_pico_W/

Raspberry Pi Pico / Pico W を使ったワークショップ用のリポジトリです。  
公開サイトの入口ページと、各企画ページ、サンプルコード、画像・動画素材をまとめています。

主な内容:
- `index.html`: 企画を選ぶ入口ページ
- `pico-heat.html` / `iot-switch.html`: 公開用の各ページ
- `single-file-examples/`: 単体で試しやすいサンプルコード
- `images/`, `videos/`: サイトや資料で使う素材
